1. Propositos

El nombre de la pagina se debe a una persona contando con sus iniciales, y fonding para hacer juego de palabras y sera lancado en la vercion 0.1 del proyecto, estara conpuesto por lenguaje html, css y JavaScript, y algo de base de datos, La primera version tendra como objetivo tener un lugar donde los artistas se puedan loguear, que los usuarios puedan interactuar, incluido los artistas.

2. Alcance del producto / Software

El alcanze que pretengo que tenga es para las personas que quieran intruducirse en el mundo Artistico, y que sino comocen mucho de internet y como es, les resulte sencillo, beneficiandolos economicamente, como objetivo principal es crear un entorno lindo esteticamente, pero a la vez simple, empezando por la creacion de usuarios que puedan loguear en la pagina y que se pueda subir archivos y sean visibles.

3. Referencias

No tome referencia de ninguna pagina todavia.

4. Funcionalidades del producto

Almacenar usuarios, subir archivos de diferentes formatos, almacenar esos archivos y creacion de perfiles

5. Clases y características de usuarios

Estara dirijida para una clase de usuario principiantes, o que tengan minimo conocimiento, haciendola un entorno amigable.

6. Entorno Operativo

Estara programado en un lenguaje html, css y JavaScript con algo de php para la base de datos, teniendo la funcion de almacenarusuarios

7. Requerimientos funcionales

7.1 Tener una feed para que los usuarios puedan ver el contenido

7.2 Perfiles unicos

7.3 Donaciones

7.3 Ver y publicar contenidos

8. Requerimientos de interfaces externas

8.1 La pagina va a ser paracomputadoras y celulares

8.2 Loa usuarios van a poder interactuar ya sea donando u publicando

8.3 La pagina interactuara con instagram, facebook o twitter

8.4 No va a interactuar con ningun software

9. Requerimientos no funcionales

Si alguien busca algo en particular se actualizara su feed automaticamente
